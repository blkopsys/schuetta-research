# A1 Release Checklist (Reusable)

## Pre-Release (T-48h)
- [ ] Final IEEE two-column PDF compiled
- [ ] 6 figures exported (PDF/PNG >= 300 dpi)
- [ ] references.bib finalized
- [ ] OSF: release folder staged (private)

## Release (T-0)
- [ ] arXiv submission completed (primary: eess.SP)
- [ ] Record arXiv ID in tracker
- [ ] OSF release created: A1-v1 (public only referenced artifacts)
- [ ] Website updated with arXiv + OSF links
- [ ] Tracker status -> Released

## Post-Release (T+24h)
- [ ] Verify public arXiv page
- [ ] Verify OSF release accessible
- [ ] Verify site renders on domain
- [ ] No further action (Scholar/ORCID auto-ingest)
